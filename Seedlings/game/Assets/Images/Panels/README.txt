ORGANIZATION AND USES

"bottom" folder:
- all generic bottom screen layouts

"variable_panels" folder:
- all associated top screens for the variables concept minigame(s)
- also now includes separated assets to build bgs individually for tutorial

"melodymak_panels"
- all associated top screens for melody maker minigame

Please refer to gameflow pdfs per game module.

NOMENCLATURE REVIEW
[game identifier or name]_[sequence number]_[subsequence number]_[description].png

ex:
mm_2_1_playScreen.png
	- mm: melody maker screen
	- 2: second screen to display in scene sequencing
	- 1: the first in a set of screens for this sequence index
	- playScreen: this screen is used when a melody is being developed or played.
